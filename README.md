# 🌱 Crop & Fertilizer Analysis Dashboard

## 📊 Overview
This project presents an interactive dashboard for analyzing the relationship between crops, fertilizers, and environmental factors such as temperature, moisture, and humidity.

## 🚀 Tools Used
- Power BI
- Data Analysis
- Data Visualization

## 📌 Key Insights
- Most Frequent Fertilizer: **Urea**
- Best Performing Crop: **Ground Nuts**
- Total Crops Analyzed: **11**
- Total Fertilizers: **7**

## 📈 Dashboard Features
- Crop distribution by soil type
- Fertilizer usage analysis
- Temperature impact on crops
- Moisture & humidity comparison
- Interactive slicers for filtering

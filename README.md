# 🌱 Crop & Fertilizer Recommendation System

### Graduation Project  

---

## 📌 Project Overview

This project focuses on analyzing agricultural data to understand the relationship between **crops, fertilizers, soil types, and environmental conditions** (temperature, humidity, and moisture).

The goal is to extract meaningful insights that help in **improving crop productivity** and **selecting the most suitable fertilizers** based on different conditions.

---

## 🎯 Objectives

* Analyze crop performance under different environmental conditions
* Identify the most frequently used fertilizers
* Determine the best-performing crops
* Study the impact of soil type on crop distribution
* Provide data-driven insights for agricultural decision-making

---

## 🛠️ Tools & Technologies

* Power BI (Data Visualization & Dashboarding)
* Excel / CSV (Data Source)
* Data Cleaning & Transformation
* Exploratory Data Analysis (EDA)

---


## 📊 Dataset Description

The dataset contains information about:

* Crop types
* Fertilizer types
* Soil categories (Clayey, Sandy, Loamy, Red, Black)
* Environmental factors (Temperature, Humidity, Moisture)

---

## 📈 Dashboard Features

### 🔹 General KPIs

* Total number of crops analyzed
* Total number of fertilizers
* Most frequent fertilizer
* Best performing crop

---

### 🔹 Visual Analysis

* Crop distribution by soil type
* Crop performance based on temperature
* Fertilizer usage across different conditions
* Moisture & humidity comparison
* Fertilizer effectiveness visualization

---

### 🔹 Interactive Elements

* Fertilizer slicer
* Crop slicer
* Dynamic filtering across all visuals

---

## 🔍 Key Insights

* **Urea** is the most frequently used fertilizer across multiple crops
* **Ground Nuts** show the best overall performance
* Soil type significantly affects crop distribution
* Higher humidity levels tend to correlate with better crop performance
* Temperature variations impact crop productivity differently
